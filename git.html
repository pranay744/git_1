Staging or Index Area: ---------------------- git commit -m --> This is push the file from Staging to the Local Repository git log --> This will show's us all commit id's list git show list all files under this commit id. Staging to Working Directory: ----------------------------- git reset HEAD -->This will bring the file from Staging area to the Working directory Local Repository to Staging Area: --------------------------------- git reset --soft --> This will brind the file from Local Repository to the Staging area Local Repository to Working Directory: --------------------------------- git mixed --> this will bring respective file from Local repository to working area along with staging file if we have any Local Repository to Central Repository: --------------------------------------- git clone --> download repository data from Central Repo to the Local system git push --> this will push all the files from local to Central Repo git pull --rebase --> this will download all remote repo files into ourlocal machine. Logs: ----- git log -n --> This will display how many number of logs you want to print. git log --oneline --> Display all logs in the single line git log --oneline -n 3 --> Display Fist 3 logs git log --author "pranay" --> Display the logs with the author name Branching: ---------- git branch --> List all branches git branch -r --> list all Central repo Branches gti branch -a --> List all local and remote Branches git branch --> create new Branch git merge master(branch) --> Copy all files from master to current branch git checkout --> Switch one branch to another branch git cherry-pick --> Copy specific Commit ID files into existing branch git push origin --> Push the branch to Central Repository Deleting Branches: ------------------ git branch -d --> Deleting branch in Local git push origin -d --> Delete Branch in Remote Repository
